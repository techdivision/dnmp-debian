# dnmp debian
All-in-One docker image based on debian.

## Services
- java v8
- elasticsearch v2.4.2
- mysql 
- nginx
- nodejs
- php v7
- postfix
- redis
- rabbitmq
- varnish (port: 81)

## PHP Xdebug 
Xdebug is disabled by default. If you want to activate it, check [docs/Xdebug/README.md](docs/Xdebug/README.md).
